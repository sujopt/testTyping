# Typing Test App

A React + TypeScript typing speed test built with Vite.

This project lets users practice typing against generated prompts and tracks live performance metrics such as words per minute (WPM), accuracy, progress, and elapsed time.

## Features

- Sentence and paragraph typing modes
- Difficulty levels: easy, medium, hard
- Live WPM, accuracy, and timer stats
- Progress bar and restart flows
- Light/dark theme toggle
- Unit tests with Vitest

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Vitest

## Getting Started

### Prerequisites

- Node.js 22+
- npm

### Install

```bash
npm ci
```

### Run in Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Run Tests

```bash
npm test
```
